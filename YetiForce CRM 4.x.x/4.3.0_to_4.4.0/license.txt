IMPORTANT !!!
Required PHP version for the update package is PHP 7.1.x. The update is not possible in case of any other version.
The upgrade package requires increasing the limits on the server because otherwise the update won't be correct.
It's necessary to install "4.3.0_to_4.4.0_PreInstall" package first because otherwise the update 4.3.0_to_4.4.0" won't be possible.

Before the update please disable all cron tasks.

Log files should be checked after update. 

Before you begin the update please check the latest server configuration requirements.
